# FPV Flight Simulator / FlightSim2
A first-person perspective flight simulator made in p5.js

Don't expect it to be amazing... the physics are pretty bad :'(

![preview.png](preview.png)

---

# Controls

Mouse > Throttle

+/- > Engine

G > Gear

F > Flaps (0-8)

B > Spoilers/Air Brakes

Space > Brakes
